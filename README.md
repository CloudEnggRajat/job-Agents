# jobAgent

A small project for scraping job listings and sending email notifications.

## Overview

- `main.py`: entry point to run the application.
- `scraper.py`: contains scraping logic for job sources.
- `email_template.py`: email content and helpers.

## Setup

1. Create and activate a virtual environment (Python 3.14 recommended).

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Configure any required credentials or environment variables before running.

## Usage

Run the main script:

```bash
python main.py
```

## Development

- Tests: add tests in a `tests/` folder and run with pytest.
